# Remote Server Manager

Create directories in the same folder as the 2 scripts.
In each folder add textdocuments for which each should hold the information for a single connection.

File structure examples:
- MaxMustermann
	- Max1
	- Max2
	- Max2
- Hades
	- Max2
	- Paul1
- Paul
	- Paul1
	- Paul2
- Main.py
-RemoteDesktopManager

Double click the RemoteDesktopManager.bat file to run Main.py

### Order of Fields for the textdocuments:
1. **Owner Name**
2. **IP Address**
3. **Password**
4. **Ingame Name**
5. **Player ID**

Example:
Max:1.1.10.127:Password123:Max_Mustermann:1234567890
